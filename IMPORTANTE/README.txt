Este projeto C# com dependências como Microsoft.OpenApi.Models (usado pelo Swagger) e Microsoft.EntityFrameworkCore, é necessário restaurar/baixar os pacotes NuGet usados no projeto.

Por isso haverão dois shell scripts dentro desta pasta, um para ser executado em sistemas Windows e outro para ser executados em Linux e MacOS.

ATENÇÃO
Sem as dependências necessárias este projeto pode não funcionar corretamente.

EXECUTE OS SHELL SCRIPTS COMO ROOT/ADMINISTRADOR!
